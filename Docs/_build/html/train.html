train module¶

Train a Classifier Deep Convolutional GAN on the Celeba dataset

train.goodfellow_loss(d_real, d_fake)¶

Ian Goodfellow loss for GAN with small epsilon for stability.

train.init_logger(name)¶

Initialize a logger with nice formatting.

train.inputs(is_training)¶

Construct input pipeline.

Use command line args to set up dataset iterators with appropriate batch sizes and epoch counts pointing to the correct tfr path. Then use is_training to return the contents of the desired iterator.

train.main(_)¶

Prepare everything and run training.

Write tfrecords if necessary. Set up TensorBoard dir and image saving folder. Write hyperparameters to file. Run training.

train.run_training()¶

Run training.

train.train(init_rate, loss, var_list, beta1=0.5, name='Train')¶

Use adam to train.
